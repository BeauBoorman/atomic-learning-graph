# Learn Atomic Learning in five small steps

A five-step tour of Atomic Learning

Atomic Learning Graph turns open educational text into a small, offline course. It breaks a hard topic into one-concept lessons, puts prerequisites in a useful order, and keeps each explanation beside the source passage that supports it.

> [!info] This is a showcase, not a generated course
> It teaches the product and lets you inspect how an Atomic Learning export should look in Obsidian.

## Start the tour

1. [[choose-a-goal|Choose a goal]] — Begin with something specific that you want to understand.

2. [[follow-the-path|Follow the path]] — Learn prerequisites before the ideas that depend on them.

3. [[learn-one-idea|Learn one idea]] — Each stop explains one concept in plain language.

4. [[check-the-receipt|Check the receipt]] — Every explanation stays close to the source passage that supports it.

5. [[take-it-with-you|Take it with you]] — Use the learning tool you already like.
