# Local teacher-builder

The local BYOK teacher-builder turns text you own into a self-contained, one-file course using your
own OpenAI, Anthropic, or OpenAI-compatible provider key. It runs on macOS, Linux, and Windows with
Node.js 22 or newer; see [the complete install, use, safety, and test-access instructions](builder/README.md).

```sh
corepack enable
pnpm install
cd builder && node server.mjs
```
